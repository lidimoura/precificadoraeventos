Este projeto é uma ferramenta de "isca digital" (lead magnet) desenvolvida como parte do **Encontro D'Água Hub**.

A calculadora, construída em Streamlit, ajuda empreendedoras da área de eventos a precificar seus serviços de forma justa e lucrativa.

## Acesso ao App

Você pode testar a calculadora ao vivo aqui:
[https://precificadoraeventos-eci5zruczxadsri5cgwuug.streamlit.app/](https://precificadoraeventos-eci5zruczxadsri5cgwuug.streamlit.app/)

## Agente de IA Integrado

Este app possui um agente de IA ("Amazô") embutido, construído com Typebot, para capturar e qualificar leads automaticamente.
